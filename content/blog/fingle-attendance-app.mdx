Portable solution for taking attendance. \
This system contains one `device` that have all hardware parts like: `ESP8266`, `Battery`, `Fingerprint Sensor`, `LEDs` etc. \
`Fingle` App will be available to admin. The app will communicate to `device` using wifi.

#### Overview

After powering on the **device** it will provide hotspot. The credentials can be changed in code statically. \
Admin will connect to the hotspot and one notification will be prompted: **Sign in to the network.** \
After clicking to the notification **Fingle App** will be launched. \

**Take Attendance Screen**

Just by openning **Take Attendance** screen system will be waiting for any enrolled finger. Just scan finger and attendance will be recorded. It's that simple. \
Details of attendee will be reflected in App.

## Technologies and hardwares:

### Client App

    - [React](https://react.dev?source=flamer "React")
    - [Web Socket API](https://developer.mozilla.org/en-US/docs/Web/API/WebSockets_API "Web-socket") for realtime communication
    - [React Context API](https://react.dev/reference/react/useContext "Context API React")
    - [Local Storage](https://developer.mozilla.org/en-US/docs/Web/API/Window/localStorage "Local storage") for storing data, can be upgraded to any database later
